# Machine Learning, Geometry Tool Image Classifier via Teachable Machine

A fully accessible, browser-based ML image classifier that identifies important geometry tools needed in real time, built to help students like me never forget about the essentials ever again.

Try the Live Demo Here: https://7s4hkp.csb.app/

## Overview

I always had trouble remembering to bring everything that I needed for math class. I would either forget one thing or the other, and it would leave me looking for alternatives. Developed at Kode With Klossy's Code-A-Bration, I modified a template model and pitched an image classifier that has been trained to recognize protractors, calculators (including the one in your phone), and pencils (both mechanical and traditional), so these essential tools will never escape your mind when packing your bag ever again.

## How to Run

1. Press the `BEGIN` button in the Preview panel to see your model in action!
2. Hold up your desired item to the camera, and watch as the image classifier identifies the item with 90% accuracy in the number panel. 

   Note: It may take a few seconds to load. Press the refresh button in the Preview panel to close the camera.

## Impact/Results

- Built ML image classifier distinguishing 3 commonly-used geometry tools with 90% classification accuracy on test data
- Independently curated and trained a 66-image dataset via Google Teachable Machine across 3 separate classes
- Iteratively improved ML model accuracy across multiple versions by diversifying training images across different angles and subcategories
- Deployed a fully interactive, browser-based web app using HTML, CSS, and JavaScript with no installation needed
- Showcased project online via Zoom to an international cohort of 10 student coders and 2 college mentors, receiving peer feedback

## Tools and Technologies

Machine Learning: Google Teachable Machine, TensorFlow.js
Frontend: HTML, CSS, JavaScript

## Key Takeaways

Through the process of building this AI/ML classifier, I learned the importance of data diversity in machine learning models and how it can impact the accuracy of my image classifier. After my model identified a compass as a pencil, I understood how the quality of data mattered more than the quantity, and made sure each image I took was unique in terms of lighting, structure, or angle. Seeing my system come together with each iteration I made had me realize why I wanted to work with AI in the first place, and I hope to keep expanding my knowledge in the application of different kinds of tech to real-world tools.
